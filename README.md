
### finite.js
JavaScript based data analysis tool.
#### Build
```
cd <to finitejs folder>
./gradlew build
```
#### Start
```
cd build/distributions/finitejs-0.1.0
./finitejs
```
#### Test
```js
var data = read.csv('samples/sample.csv');
print(data);
quit();
```
